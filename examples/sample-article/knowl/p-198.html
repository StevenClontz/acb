Paragraph

Everybody wants the ampersand, it is the most-dangerous character. It is the escape character for XML, and LaTeX uses it to organize tables and arrays, and for aligning equations. Consistently use the element <ampersand /> to make a literal ampersand in normal text, such as in “A&P.” In mathematics, and other places where you are using LaTeX syntax, use the pre-defined \amp macro. For code listings and other verbatim text, use the escaped XML entity &amp;.
